## Product Microservice

Clone

```bash
https://github.com/NestJs-Microservices-Gior/products-microservices.git
```

Install dependencies

```bash
$ npm install
```

Copy file `.dev` and rename `.dev.template`

Execute Prisma migration `npx prisma migrate dev`

Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
